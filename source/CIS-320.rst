Assignments
^^^^^^^^^^^

Because this is the first time this class has been taught (web development used
to be one semester) the assignments below may change. We'll see how the class
progresses.

Assignments will involve a 10 point quiz and a 20 point lab. The exceptions will
be Assignment 7 and Assignment 13. These are written reports that will be worth
30 points (Assignment 7) and 50 points (Assignment 13).

* Lab 1: Programming Environment Setup Assignment
* Lab 2: JavaScript Assignment
* Lab 3: List DB Records in JSON Format Assignment
* Lab 4: List Records - Final Assignment
* Lab 5: Form Validation Assignment
* Lab 6: Insert a Record Assignment
* Lab 7: Security Writing
* Lab 8: Back-End Validation and Deleting Records
* Lab 9: Editing a record
* Lab 10: Login Assignment
* Lab 11: Final Lab
