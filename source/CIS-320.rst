Assignments
^^^^^^^^^^^

Because this is the first time this class has been taught (web development used
to be one semester) the assignments below may change. We'll see how the class
progresses.

Assignments will involve a 10 point quiz and a 20 point lab. The exceptions will
be Assignment 7 and Assignment 13. These are written reports that will be worth
30 points each.

* Monday, Jan 16 at 8 am: Assignment 1 - Create your own web development stack
* Monday, Jan 23 at 8 am: Assignment 2 - Submit and validate a form via JavaScript
* Monday, Jan 30 at 8 am: Assignment 3 - Insert new records in a database
* Monday, Feb 06 at 8 am: Assignment 4 - Update and delete records in a database
* Monday, Feb 13 at 8 am: Assignment 5 - Add the ability to log in
* Monday, Feb 20 at 8 am: Assignment 6 - Hashing and Encryption
* Monday, Feb 27 at 8 am: Assignment 7 - Research and write a plan for doing
  security testing for a website
* Monday, Mar 06: Spring Break - No assignment
* Monday, Mar 13 at 8 am: Assignment 8 - Update the "Create Read Update Delete"
  application to use JavaScript, jQuery, and JSON
* Monday, Mar 20 at 8 am: Assignment 9 - Create an graphing application that
  uses the D3 library
* Monday, Mar 27 at 8 am: Assignment 10 - TBD
* Monday, Apr 03 at 8 am: Assignment 11 - TBD
* Monday, Apr 10 at 8 am: Assignment 12 - TBD
* Monday, Apr 17: Work on final project
* Monday, Apr 24 at 8 am: Assignment 13 - Final report, write up a tutorial on
  how to use a new web technology (**TODO:** Update with final time)
